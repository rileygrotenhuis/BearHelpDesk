Submit Ticket Page:
    - POST '/submit/ticket' => Creates a new IT Support Ticket and add it to the database

Employee Login Page:
    - POST '/login' => Authenticate a user with a given email and matching password
    - GET '/login' => Get the user information of the authenticated user
    - DELETE '/logout' => Logout an authenticated user

Employee Dashboard Page:
    - GET '/dashboard/:employeeEmail/tickets/current' => Get all of the tickets that an employee is currently working on
    - PUT '/dashboard/ticket/:ticketID/status' => Update the status of a given ticket
    - POST '/dashboard/contact/:clientEmail' => Send an email to the client with their given email address
    - DELETE '/dashboard/ticket/:ticketID' => Delete a given ticket from the database

Employee Profile Page:
    - GET '/profile/employee/:employeeEmail' => Get all of the personal information of a given employee
    - PUT '/profile/employee/:employeeEmail' => Update the personal information of a given employee
    - GET '/profile/:employeeEmail/tickets/completed' => Get all of the tickets that an employee has completed

Ticket Board Page:
    - GET '/board/tickets/all' => Get all of the tickets that no employee is currently workign on
    - PUT '/board/ticket/:ticketID/assign' => Changes the employee who is currently working on a given ticket
