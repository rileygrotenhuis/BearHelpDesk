# Bear Help Desk

Bear Help Desk is an IT Support Ticket Management System where users can submit IT Support Tickets and Employees can access, manage, and work on those submitted tickets.

## Installation

You will need to have the latest version of `node` and `npm` to run this project:
- [Node](https://nodejs.org/en/download/)
- [NPM](https://www.npmjs.com/get-npm)

You will also need to download the source code using the following [link](https://github.com/rileygrotenhuis/BearHelpDesk/archive/refs/heads/master.zip)


## Usage

Inside of one terminal/command prompt window, run the following:
``` bash
cd BearHelpDesk
cd server
npm install
npm run dev
```
This will start the server side of the project (Node Backend API)

Inside of another terminal/command prompt window, run the following:
``` bash
cd BearHelpDesk
cd client
npm install
npm start
```
This will start the client side of the project (React Frontend)

Then you can open your favorite web browser and go to `http://localhost:3000` to begin using Bear Help Desk for yourself.
